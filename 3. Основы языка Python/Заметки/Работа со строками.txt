____________
Строка str
____________

- Состоит из набора символов
- Индексы начинаются с 0
- Можно получить символ по индексу: friend[1] - означает 2й символ строки
- В Python можно использовать отрицательные индексы: 
friend[-2] - означает 2й символ с конца строки и т.д.

_____________________________
Функция len и методы строки
_____________________________

- len(friend) - длина строки (сколько в ней символов)
- friend.find(‘a’) - ищем символ ‘a’ в строке
- friend.split() - разбиение строки через пробел 
- friend.isdigit() - строка состоит из чисел
- friend.upper() - приведение строки к верхнему регистру
- friend.lower() - приведение строки к нижнему регистру

Срезы
_______

- Можно получить несколько символов (часть строки) 
- friend[начальный индекс:конечный индекс]:

friend[1:4] - 1 - с какого символа, 4 - по какой символ
friend[:4] - срез с начала строки до 5го символа
friend[1:] - срез от 2го сивола до конца строки
