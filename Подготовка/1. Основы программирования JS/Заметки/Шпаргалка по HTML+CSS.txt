<title>...</title> - Содержит название вкладки.

<h1>...</h1> тег используется для заголовка. Чем больше цифра, тем мельче шрифт.

<a href="ссылка">...</a> 
<a>...</a>- от слова Anchor(Якорь) тег создания ссылки для переходов. 
href="ссылка" - от слова Hyper Reference(Гиперссылка) - атрибут тега <a>, его значение равняется необходимой ссылке.
... - указываем текст в котором будет содержаться ссылка.

<img src="img/circle+.png"> - (пример) - тег для вставки картинок/фото. Атрибут src="..." - содержит путь к картинке.

<p>...</p> - тег параграфа, создает абзац.

<b>...</b> - от слова bold (жирный...) - Тег выделяющий текст жирным.

<i>...</i> - от слова italics (курсив) - Тег выделяющий текст курсивом. 

<br> - перенос строки.

Copyright &copy; Evgeny Kirillin. - значок копирайт.

<body style="background-color: green"> - (пример) - указываем что стиль заднего плана тега body(тела сайта) вот такой то.
					Атрибут style="..." можно использовать в любом теге, где можно изменить цвет фона, текста и т.д.

<div></div> - группирует элементы (например вверху сайта текст с гиперссылками, кнопки.

							//////CSS - cascading style sheet//////

Если мы хотим применить стиль глобально, + улучшить читаемость кода, то создаем в папке с сайтом файл с расширением .css. 
В нем будут содержаться все необходимые нам стили.
Что бы привязать файл со стилями к сайту нужно использовать специальный тег. Пишется он внутри тега <head>...</head>.
<link rel="stylesheet" href="style.css"> - ТЕГ НАЗЫВАЕТСЯ <link...>, используемые атрибуты:
rel="..." <-- указываем, что тип этого документа stylesheet 
href="..." <-- указываем название файла


div{
	background-color: #dfdfdf; <-- цвет заднего плана рамочки сгруппированного элемента (по сути рамка/окантовка рамки).
	border-width: 0.14px; <-- ширина окантовки рамки. 
	border-color: #b5b7ba; <-- цвет окантовки рамки.
	border-style: solid; <-- стиль окантовки рамки.
	width: 1520px; <--  ширина самой рамки.
	height: 68px;  <-- высота самой рамки.
	padding-top: 20px; <-- отступ сверху. (padding-botom и т.д.) Пиксели отступа желательно вычитать из пикселей высоты 
												(68-20 = 48) будет тогда высота
}
